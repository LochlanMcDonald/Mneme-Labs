# Mneme-Labs

## Projects

- **Groundwork** ([`groundwork/`](groundwork/)): a web app that gives
  startups a tailored cybersecurity roadmap. A short onboarding wizard learns
  what the company does, what data it handles, and what systems it runs, then
  a rules engine builds a phased, plain-language security plan (baseline
  essentials plus profile-specific controls) with progress tracking and
  Markdown export. React + TypeScript + Vite.

- **StoveWatch** ([`stove-monitor/`](stove-monitor/)): a clip-on Pi camera
  that learns what your stove looks like when it's off, checks a snapshot
  every 5 minutes, and pushes an alert to the StoveWatch iOS app whenever the
  stove or oven appears to be in use. Includes the device agent, FastAPI
  backend, and SwiftUI app.
